## Overview

Idea Title: AI answer scout for businesses competing block by block
Product Name (proposed): Answerspot

### Core Concept

AI-powered search replaced the 10 blue links. Homeowners type "best roofer" into a chat assistant and get 3 names back. Reviews, citations, and how a company's details are marked up across the web decide which companies appear in the answer. Roofers, dentists, HVAC contractors, and auto shops compete block by block for those slots. Only 1.2% of local firms show up.
